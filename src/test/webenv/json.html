Json Sample


"Json Syntax" {


	json     = value | object | array .


	object = "{" identifier "}" .


	expression = term { "|" term } .


	term       = factor { factor } .


	factor     = identifier
	           | literal
	           | "[" expression "]"
	           | "(" expression ")"
	           | "{" expression "}" .


	identifier = character { character } .


	title      = literal .


	comment    = literal .


	literal    = "'" character { character } "'"
	           | '"' character { character } '"' .


} "The json spec noted down as rail-road-diagram made its use very popular in the web-context."

This shows pretty much the same diagrams as found on json.org except here they are generated on the fly by ebnfdiagram.
